# Venko Backend (prueba tecnica)

Backend desarrollado con Spring Boot para gestión de proyectos.

## Tecnologías
- Java
- Spring Boot
- Spring Security con JWT
- Spring Data JPA
- H2 Database (en memoria)
- Swagger para documentación

## Instalación
Clonar el proyecto

Ejecutar con Maven:

```bash
mvn spring-boot:run
